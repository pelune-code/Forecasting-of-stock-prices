Work: Forecasting of stock prices
- Author: Peter Lukáš Nedbálek
- Supervisor: Mgr. Katarína Lučivjanská, PhD.

This project consists of implementation of 4 technical indicators:
- SMA-Crossover (Simple Moving Average-Crossover)
- MACD (Moving Average Convergence/Divergence)
- Bollinger Bands: The first approach
- Bollinger Bands: The second approach
  
The zip file consists of jupyter notebook investing_in_stocks_nedbalek.pynb as long as the jupyter notebook is big in size. 
Download it, extract it and then open the notebook in an appropriate environment (Google Colaboratory, 
Anaconda, ...).
